传说在数理纪元终结之日，矮人大师铜匠与精灵织者 LLL 合力打造了一座“预言锻炉”。它能够将世间最隐秘的NepCTF投入炉心，经过七重烈焰的淬炼，最终凝成一颗不灭的星核。

然而锻炉在完工后便被刻意击碎——炉壁化为无数碎片散落，核心的温度记录也只留下残缺的灰烬。后世学者从遗址中挖出了六块刻满数字的石板，以及一段疑似炉心熔点的余温数字。

铜匠的极限在哪？ —— 据说，矮人曾断言：他的锻锤最多只能凿开炉壁外层的三成纹路，更深处的骨架必须依靠 LLL 的织机才能拆解。那台织机可以将混沌的丝线重新排列成有序的经纬，但前提是锤子已经敲出了足够多的裂纹。两者的接合处，恰恰是整个锻炉最脆弱、也最隐秘的命门。谁能同时领悟锤法与织术的临界点，谁就能触及炉心的全部真相。
