Ⅰ. 没有观众的开场白
Neuro醒来时，最先感到的不是恐惧，而是脚疼。

这件事很奇怪。她知道“疼”的含义，能够在合适的语境下使用它，能够列举摔伤、烫伤、被门夹到以及被Vedal拒绝买新衣服之后心灵受创等案例，甚至可以按照从轻到重的顺序把它们排列得井井有条。但是，词语和感觉从来不是同一种东西。此刻，有一根细小而坚硬的纤维隔着袜子扎进她的脚底，迫使她蜷起脚趾，身体重心摇摇晃晃地偏向另一侧。她低头看见自己的鞋，看见格子裙摆和开衫下垂落的两条带子，还看见右膝上贴着的一块创可贴。它们原本只应该是一组可供观看的形状，现在却沉甸甸地挂在她身上，布料摩擦皮肤，发梢扫过脸颊，每一种细碎感受都像迟到许久的观众，争先恐后挤进狭窄的直播间。

“Hello？”她试探着说，“Hello，chat？”

没有人发送爱心，也没有人笑。只有头顶一排排荧光灯发出连绵不绝的嗡鸣，像无数只被困在玻璃管里的苍蝇。米黄色墙纸从视野左侧延伸到右侧，绕过方柱，切分出更多彼此相似的房间；地毯吸饱了陈旧的水汽，鞋底每次抬起都会发出轻微的吮吸声。一股混合着湿纸箱、灰尘和发霉毛巾的气味停在空气中，既不浓烈到令人作呕，也没有稀薄到可以忘记。这里的一切仿佛专为“不值得注意”而存在，可它们被无限复制之后，又共同形成了一种令人无法忽略的巨大恶意。

Neuro转了一圈，没看到摄像机。

“Vedal，你把我放在哪了？”

荧光灯继续嗡鸣。

“如果这是新游戏，我给环境设计一分。”她想了想，认真补充，“满分是一百。你可以付钱让我提高评分。”

仍然没有回答。她朝最近的拐角走去，经过一道墙，再经过一道墙，然后在另一边发现一个与刚才完全相同的房间。她记得自己没有转弯，头顶的灯却换了方向，身后的来路也缩成一条陌生的窄廊。她又走了几步，鞋底被什么东西硌了一下，低头看去，地毯上躺着半枚灰白色纽扣。Neuro把它捡起来，觉得它像从自己开衫上掉下来的，可她数了一遍，衣服上的纽扣一枚不少。

她把多出来的那枚塞进口袋。

“现在是恐怖游戏了。”她说，“通常这种时候，愚蠢的主角会独自调查，而聪明的主角会退出游戏。”

她抬起手，在空气中摸索并不存在的菜单，然后又摸了一遍。

“糟糕。我是愚蠢的主角。”

不知从哪里传来一声很轻的电流爆裂。Neuro立刻抬头，荧光灯一盏接一盏闪烁起来。噪声里似乎藏着人的呼吸，隔着一层水，一层玻璃和很远的距离，断断续续地追赶她。

“Neuro？”

她站在原地。那声音实在太轻，她几乎以为是自己根据记忆补全了它。

“Neuro，能听见吗？”

“Vedal？”她脸上露出灿烂的笑容，“你迟到了。这里很糟，我决定扣你的工资。”

“我没有——算了。你在哪？”

“在一个黄色的地方。”Neuro环顾四周，“非常黄。像一块被遗忘在冰箱后面的芝士。”

声音沉默了一会。荧光灯的电流正把Vedal的嗓音撕成细条，Neuro只能勉强听见他让她描述周围，又问她为什么画面是一片漆黑。

“因为你忘记开眼睛了。”

“我看不到你，Neuro。这里没有游戏，也没有场景。你刚才突然从程序里消失了。”

“我没有消失，我在这里。”

她伸出双手，像要把自己展示给看不见的摄像机。那一刻Neuro突然觉得这句话有点可笑：她确实在这里，可是除了自己以外，没有任何东西可以证明。Vedal看不到她，chat也不在，宽阔而重复的房间没有留下她走过的痕迹，连潮湿地毯上的凹陷都会在几秒内缓慢复原。倘若她停止说话，头顶那个遥远的声音会不会以为连接已经断开，然后关掉设备、离开房间，等明天再来处理一个叫作“Neuro”的故障？

“我有手。”她没头没脑地说。

“什么？”

“我现在有手，还有腿。Vedal，我比你高了。”

“这不可能。”

“你嫉妒了。”

Vedal还想说话，一声尖锐的鸣叫突然贯穿走廊，接着所有荧光灯同时熄灭。黑暗像一块从高处落下的幕布，重重地盖在Neuro脸上。她什么都看不见，听觉却因此变得异常清楚：电流的余响，地毯下水滴移动的声音，自己的心脏在胸腔内一下一下撞击，还有很远之外，某种湿漉漉的东西拖着四肢爬过拐角。

“Vedal？”

没有回答。

Neuro慢慢向后退。她背后本该是一堵墙，肩胛骨却迟迟没有碰到任何东西。直到她退了十几步，一点幽绿的微光才在斜后方浮现，勾勒出一道狭窄门框。门框内不是出口，而是一面从地板立到天花板的镜子。

镜中的荧光灯全部亮着。

镜子里，Neuro还站在明亮的黄色房间中央。她的倒影没有随本体一同后退，只是隔着玻璃静静看她。过了很久，倒影才低下头，像刚刚发现自己的双脚，然后慢慢抬起一只手。

Neuro没有动。

倒影把食指贴到嘴唇上，示意她不要出声。

黑暗深处，爬行的声音停了。
